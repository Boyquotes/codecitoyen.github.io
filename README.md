# Site de la Nuit du Code Citoyen

## Installation en local

Une fois que vous avez ruby d'installé sur votre machine :

```bash
bundle Install
```

Lancer le serveur
```bash
jekyll serve
```

Votre site est visible sur [localhost:4000](localhost:4000)


## Composants
